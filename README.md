# Nostra-Web-page
Nostra is a responsive fashion e-commerce website built using HTML, CSS, and JavaScript.** **It features product collections, search, filters, and a modern user-friendly design.Nostra is a modern and responsive fashion e-commerce website developed using HTML, CSS, and JavaScript. The project provides a clean shopping interface with product collections, search functionality, category filtering, and interactive elements.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern fashion store user interface
- Home page with banner slider and product sections
- Collections page with search and filter options
- Contact page with user-friendly form
- Product cards with images and details
- Smooth navigation between pages

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

- `index.html` - Home page
- `collections.html` - Product collections page
- `contact.html` - Contact page
- `style.css` - Website styling
- `script.js` - Interactive functionalities

## Purpose

This project was created to practice front-end web development concepts and build a responsive e-commerce website using basic web technologies.
